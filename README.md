Feedstock for [yadism](https://yadism.readthedocs.io) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Yet Another DIS Module

```bash
conda install -c hep-forge -c conda-forge yadism
```
